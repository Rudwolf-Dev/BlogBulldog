
# 🐾 Blog Bulldog

Sitio web institucional desarrollado para difundir actividades, clubes y equipos representativos de la escuela.  
El proyecto fue realizado en **equipo de 3 personas**, aplicando buenas prácticas de desarrollo colaborativo y despliegue en la nube.

## 👥 Roles
- **Rudwolf-Dev (Rodolfo)** → Desarrollo principal: montaje del código base, integración de componentes, configuración de Vite, ESLint, Prettier, pruebas y despliegue en Vercel.
- **tinaenigma** → Diseño visual y estilos, asegurando una interfaz atractiva y coherente.
- **euclsin** → Redacción y organización del contenido informativo.

---

## 🚀 Tecnologías utilizadas
- **[Vue 3](https://vuejs.org/)** – Framework principal
- **[Vite](https://vitejs.dev/)** – Bundler y servidor de desarrollo
- **[Pinia](https://pinia.vuejs.org/)** – Manejo de estado global
- **[Bootstrap 5](https://getbootstrap.com/)** + **Bootstrap Icons** – Estilos y componentes
- **[Vitest](https://vitest.dev/)** – Pruebas unitarias
- **[Nightwatch](https://nightwatchjs.org/)** – Pruebas end-to-end
- **[Vercel](https://vercel.com/)** – Despliegue automático desde GitHub

---

## ⚙️ Scripts principales
- `npm run dev` → Inicia el servidor de desarrollo con Vite  
- `npm run build` → Compila el proyecto para producción  
- `npm run preview` → Previsualiza la build de producción  
- `npm run test:unit` → Ejecuta pruebas unitarias con Vitest  
- `npm run test:e2e` → Ejecuta pruebas end-to-end con Nightwatch  
- `npm run lint` → Corre linters (Oxlint + ESLint) para mantener el código limpio  
- `npm run format` → Formatea el código con Prettier  

---

## 📦 Instalación y uso
Clona el repositorio:
```bash
git clone https://github.com/Rudwolf-Dev/BlogBulldog.git
```

Instala dependencias:
```bash
npm install
```

Ejecuta en modo desarrollo:
```bash
npm run dev
```

Compila para producción:
```bash
npm run build
```
